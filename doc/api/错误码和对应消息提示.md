> 如果成功则返回固定code和message
>
> code: 200
>
> message: success

>错误码格式：xxxx
>
>前两位表示错误模块，后两位标识错误类型

| code |        message        | describe  |
| :--: | :-------------------: | :-------: |
| 1001 |     身份验证失效      | token失效 |
| 1002 | token缺失，请重新登录 |           |
|      |                       |           |
